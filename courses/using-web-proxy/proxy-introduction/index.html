WebProxy Bilan Tanishuv

Bugungi kunda ko'pgina zamonaviy veb va mobil ilovalar ma'lumot yuborish-qabul qilish uchun doimiy ravishda back-end serverlarga ulanadi va ushbu ma'lumotni brauzer yoki telefonida qayta ishlaydi. Ko'pchilik ilovalar ma'lumotni qayta ishlashda back-end serverlarga kuchli tayanar ekan, aynan shu serverlarni testlash va himoyalash tobora muhimlashib bormoqda.

Back-end serverlarga yuboriladigan veb so'rovlarni testlash Web Application Penetration Testing ishlarining asosiy qismini tashkil etadi va bu g'oyalar ham veb, ham mobil ilovalarga tegishli. Ilova bilan back-end o'rtasidan o'tadigan so'rov/traffikni ushlash va test maqsadida ularni o'zgartirish uchun Web Proxies dan foydalanishimiz kerak bo'ladi.

// Web Proxies nima?

Web proxies — brauzer yoki mobil ilova bilan back-end server orasiga qo'yiladigan maxsus vositalar bo'lib, ular ikki tomon orasida ketayotgan barcha veb so'rovlarni ushlab ko'rish imkonini beradi; qisqasi, man-in-the-middle turidagi vositalar sifatida ishlaydi.

Boshqa Network Sniffing ilovalari masalan, Wireshark tarmoqdan o'tayotgan barcha lokal trafikni tahlil qilish orqali ishlasa, Web Proxies asosan HTTP/80 va HTTPS/443 kabi veb portlar bilan ishlaydi.

Web proxies har qanday veb pentester uchun eng zarur vositalardan hisoblanadi. Ular CLI-asosidagi oldingi vositalarga nisbatan veb so'rovlarni ushlash va qayta yuborishni ancha soddalashtiradi. Web proxy sozlangach, ilova yuborgan barcha HTTP so'rovlar va back-end serverdan kelgan javoblarni ko'ra olamiz. Bundan tashqari, muayyan so'rovni intercept qilib, undagi ma'lumotni o'zgartirib, back-end buni qanday qayta ishlashini sinab ko'rishimiz mumkin — bu har qanday veb pentestning muhim qismidir.

// Web Proxies dan foydalanish sohalari

Asosiy vazifa — HTTP so'rovlarni ushlash va qayta yuborish — dan tashqari, web proxies ko'plab boshqa imkoniyatlarga ham ega. Quyidagilar ulardan ba'zilari:

  • Veb ilovalarda zaifliklarni skanerlash
  • Web fuzzing
  • Web crawling
  • Web ilovani xaritalash
  • Web so'rovlarni tahlil qilish
  • Web konfiguratsiyalarni sinash
  • Code review-larni yengillashtirish

Ushbu modulda biz aniq veb hujum turlarini muhokama qilmaymiz. Biroq web proxies dan foydalanish va ularning turli funksiyalarini batafsil ko'rib chiqamiz hamda qaysi veb hujum turi uchun qaysi funksiyalar zarurligini eslatib o'tamiz. Biz ikkita eng ommabop web proxy vositasini yoritamiz: Burp Suite va ZAP.

// Burp Suite

Burp Suite (oddiy qilib aytsak "Burp Sweet" deb talaffuz qilinadi) — veb pentesting uchun eng ommabop veb proxy hisoblanadi. U keng funksiyalar uchun qulay UI ga ega va hatto ichki Chromium brauzerini ham taqdim etadi. Ba'zi Burp funksiyalari faqat tijoriy Burp Pro/Enterprise versiyalarida mavjud, biroq bepul community versiyasi ham juda kuchli test vositasi hisoblanadi.

Pullik-gina funksiyalar

  • Active web app scanner
  • Tezkor Burp Intruder
  • Muayyan Burp Extensions ni yuklash imkoniyati

Community (bepul) versiya ko'pchilik pentesterlar uchun yetarli bo'ladi. Ilg'orroq veb pentestlarga o'tganda, pro funksiyalar qo'l kelishi mumkin. Ushbu modulda ko'rib chiqiladigan funksiyalarning aksariyati community versiyada mavjud; shu bilan birga, Active Web App Scanner kabi pro funksiyalariga ham qisqacha to'xtalamiz.

Maslahat: Agar sizda ta'lim yoki biznes email manzilingiz bo'lsa, keyinroq ko'rsatilgan maxsus havola orqali Burp Pro'ning bepul sinov muddatiga ariza topshirib, ushbu modulning pro-gina funksiyalarini ham amalda kuzatib borishingiz mumkin.

// OWASP Zed Attack Proxy

OWASP Zed Attack Proxy — veb pentesting uchun yana bir keng tarqalgan veb proxy vositasi. ZAP — OWASP tomonidan boshlangan va hamjamiyat tomonidan qo'llab-quvvatlanadigan bepul va open-source loyiha, shuning uchun Burp'dagi kabi "pullu-gina" funksiyalar yo'q. So'nggi yillarda ZAP jadal rivojlanib, open-source veb proxy vositalari ichida yetakchi mavqega chiqib bormoqda.

Burp singari, ZAP ham veb pentestingda qo'l keladigan bir qator asosiy va ilg'or funksiyalarga ega. ZAP'ning ayrim ustun tomonlari bor — ular haqida modul davomida gaplashamiz. ZAP'ning asosiy afzalligi — to'liq bepul va open-source ekani: demak, skanerlarda pullik obuna bilan ochiladigan cheklovlar yo'q. Bundan tashqari, hamjamiyat kengayib borar ekan, ZAP ko'plab "Burp'da pullik-gina bo'lgan" funksiyalarni ham bepul taklif etib bormoqda.

// Xulosa

Har ikkala vositani o'rganish jarayoni ancha o'xshash va pentest davomida turli vaziyatlar uchun tanlov beradi — qaysi biri ehtiyojimizga mos bo'lsa, o'shani qo'llaymiz.

Ba'zan Burp obunasining qiymati o'zini oqlamasligi mumkin va to'liq ochiq hamda bepul tajriba uchun ZAP'ga o'tish ma'qul bo'ladi. Boshqa holatlarda esa, yirik yoki korporativ pentestlar uchun pishiqroq yechim zarur bo'lsa, Burp Pro taqdim etadigan qiymat o'zini oqlaydi va shu funksiyalar uchun Burpdan foydalanish mantiqli bo'ladi.
